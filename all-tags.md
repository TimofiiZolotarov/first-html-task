# Таблиця HTML тегів

## Базові структурні теги

| Тег | Функція |
|-----|---------|
| `<!DOCTYPE html>` | Оголошення типу документа HTML5 |
| `<html>` | Кореневий елемент HTML документа |
| `<head>` | Контейнер для метаданих документа |
| `<body>` | Основний контейнер для видимого вмісту сторінки |

## Метадані та підключення ресурсів

| Тег | Функція |
|-----|---------|
| `<meta>` | Визначає метадані (charset, viewport, description, Open Graph) |
| `<title>` | Встановлює заголовок сторінки у вкладці браузера |
| `<link>` | Підключає зовнішні ресурси (CSS, іконки) |
| `<script>` | Підключає або виконує JavaScript код |

## Семантичні секції

| Тег | Функція |
|-----|---------|
| `<header>` | Шапка сторінки або секції |
| `<main>` | Основний контент сторінки |
| `<aside>` | Бічна панель або додатковий контент |
| `<nav>` | Навігаційна секція |
| `<article>` | Самостійна стаття або контентна одиниця |

## Заголовки

| Тег | Функція |
|-----|---------|
| `<h1>` | Заголовок першого рівня |
| `<h2>` | Заголовок другого рівня |
| `<h3>` | Заголовок третього рівня |
| `<h5>` | Заголовок п'ятого рівня |

## Текстові елементи

| Тег | Функція |
|-----|---------|
| `<p>` | Параграф тексту |
| `<strong>` | Виділення тексту жирним шрифтом (важливий текст) |
| `<pre>` | Попередньо відформатований текст (зберігає пробіли та переноси) |
| `<code>` | Фрагмент коду |

## Списки

| Тег | Функція |
|-----|---------|
| `<ul>` | Ненумерований список |
| `<ol>` | Нумерований список |
| `<li>` | Елемент списку |

## Таблиці

| Тег | Функція |
|-----|---------|
| `<table>` | Таблиця для структурованих даних |
| `<thead>` | Заголовок таблиці |
| `<tbody>` | Тіло таблиці з основними даними |
| `<tr>` | Рядок таблиці |
| `<th>` | Комірка заголовка таблиці |
| `<td>` | Звичайна комірка таблиці |

## Інші елементи

| Тег | Функція |
|-----|---------|
| `<div>` | Універсальний блоковий контейнер для групування елементів |
| `<a>` | Гіперпосилання для навігації |
| `<hr>` | Горизонтальна лінія-роздільник |

# Bootstrap класи
## Колірні класи (Background & Text)

| Клас | Функція |
|------|---------|
| `bg-light` | Світлий фоновий колір для body |
| `bg-dark` | Темний фоновий колір для header |
| `text-white` | Білий колір тексту |

## Layout (Сітка і контейнери)

| Клас | Функція |
|------|---------|
| `container-fluid` | Контейнер на всю ширину екрану з відступами |
| `row` | Рядок для Bootstrap сітки |
| `col-lg-3` | Колонка шириною 3 з 12 на великих екранах (≥992px) |
| `col-lg-9` | Колонка шириною 9 з 12 на великих екранах (≥992px) |

## Відступи (Spacing)

| Клас | Функція |
|------|---------|
| `py-4` | Вертикальні відступи (padding top & bottom) розміру 4 |
| `mb-0` | Нульовий нижній margin |
| `mb-2` | Нижній margin розміру 2 |
| `mb-4` | Нижній margin розміру 4 |

## Компоненти Card

| Клас | Функція |
|------|---------|
| `card` | Контейнер-картка з рамкою та тінню |
| `card-body` | Внутрішній контент картки з відступами |
| `card-title` | Заголовок картки |

## Таблиці

| Клас | Функція |
|------|---------|
| `table` | Базовий стиль таблиці Bootstrap |
| `table-bordered` | Додає рамки до всіх комірок таблиці |
| `table-striped` | Чергування кольору рядків (зебра-стиль) |
| `table-dark` | Темний фон для секції таблиці |

## Списки

| Клас | Функція |
|------|---------|
| `list-unstyled` | Прибирає маркери списку та стандартні відступи |

## Позиціонування

| Клас | Функція |
|------|---------|
| `sticky-lg-top` | Фіксує елемент зверху при прокрутці на великих екранах (≥992px) |

## Текстові утиліти

| Клас | Функція |
|------|---------|
| `text-decoration-none` | Прибирає підкреслення у посилань |
